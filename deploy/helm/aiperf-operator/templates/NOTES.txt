AIPerf Operator has been installed!

1. Verify the operator is running:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "aiperf-operator.name" . }}

2. Create a benchmark job:
   cat <<EOF | kubectl apply -f -
   apiVersion: aiperf.nvidia.com/v1alpha1
   kind: AIPerfJob
   metadata:
     name: my-benchmark
   spec:
     userConfig:
       endpoint:
         url: http://your-inference-server:8000
         model_names: [your-model]
         type: openai
       loadgen:
         concurrency: 10
         request_count: 100
   EOF

3. Monitor the benchmark:
   kubectl get aiperfjob my-benchmark -w

4. View detailed status:
   kubectl describe aiperfjob my-benchmark

{{- if .Values.storage.enabled }}

5. Access stored results:
   Results are stored at: {{ .Values.storage.mountPath }}/<job-id>/

   To copy results locally:
   kubectl cp {{ .Release.Namespace }}/{{ include "aiperf-operator.fullname" . }}-<pod-suffix>:{{ .Values.storage.mountPath }}/<job-id>/ ./results/
{{- end }}
