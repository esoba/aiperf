AIPerf Operator has been installed!

1. Verify the operator is running:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "aiperf-operator.name" . }}

2. Create a benchmark job:
   cat <<EOF | kubectl apply -f -
   apiVersion: aiperf.nvidia.com/v1alpha1
   kind: AIPerfJob
   metadata:
     name: my-benchmark
   spec:
     image: nvcr.io/nvidia/aiperf:latest
     models:
       - your-model
     endpoint:
       urls:
         - http://your-inference-server:8000/v1/chat/completions
       type: chat
       streaming: true
     datasets:
       main:
         type: synthetic
         entries: 100
         prompts:
           isl: 128
           osl: 128
     load:
       profiling:
         type: concurrency
         concurrency: 10
         requests: 100
   EOF

3. Monitor the benchmark:
   kubectl get aiperfjob my-benchmark -w

4. View detailed status:
   kubectl describe aiperfjob my-benchmark
