AIPerf Operator has been installed!

1. Verify the operator is running:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "aiperf-operator.name" . }}

2. Create a benchmark job:
   cat <<EOF | kubectl apply -f -
   apiVersion: aiperf.nvidia.com/v1alpha1
   kind: AIPerfJob
   metadata:
     name: my-benchmark
     namespace: {{ .Values.benchmarkNamespace.name }}
   spec:
     # Deployment fields use camelCase (K8s convention)
     image: {{ .Values.defaults.image }}

     # Benchmark config lives under spec.benchmark in snake_case
     # (identical to AIPerf CLI YAML config files)
     benchmark:
       models:
         items:
           - name: your-model
       endpoint:
         urls:
           - http://your-inference-server:8000
         type: chat
   EOF

3. Monitor with the live dashboard:
   aiperf kube watch my-benchmark

   Or with kubectl:
   kubectl get aiperfjob my-benchmark -n {{ .Values.benchmarkNamespace.name }} -w

4. View detailed status:
   kubectl describe aiperfjob my-benchmark -n {{ .Values.benchmarkNamespace.name }}

5. Retrieve results:
   aiperf kube results my-benchmark

{{- if .Values.storage.enabled }}

   Results are also stored on the operator PVC at: {{ .Values.storage.mountPath }}/<job-id>/
{{- end }}

6. Run preflight checks before submitting:
   aiperf kube preflight my-config.yaml

For more information, visit: https://github.com/nvidia/aiperf
